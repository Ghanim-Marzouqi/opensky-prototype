## 1. Executive Summary

### 1.1 Purpose
Build a comprehensive Finance module tailored for OpenSky International's wholesale import/export business, managing multi-currency transactions, invoicing, payments, and financial reporting with full audit trail support.

### 1.2 Target Users
| User Role | Primary Needs |
|-----------|--------------|
| **Finance Manager** | Full financial control, reporting, approvals |
| **Accountant** | Day-to-day transactions, reconciliation |
| **Accounts Receivable** | Customer invoices, payment collection |
| **Accounts Payable** | Supplier invoices, payment processing |
| **Executive** | Financial dashboards, KPIs, reports |
| **Operations** | View invoices, costs related to orders |

### 1.3 Business Value
- Real-time visibility into financial position
- Multi-currency support for international trading (USD, EUR, INR, CNY, OMR)
- Streamlined invoice-to-payment workflows
- Automated aging and collection tracking
- Comprehensive audit trail for compliance
- Integration with Sales and Purchasing modules

## 4. Data Models

### 4.1 Invoice Entity
```typescript
interface Invoice {
  id: string;
  invoiceNumber: string;         // INV-2024-0001
  type: 'invoice' | 'credit_note';
  
  // Parties
  customerId: string;
  customerName: string;          // Denormalized for display
  
  // Dates
  invoiceDate: Date;
  dueDate: Date;
  
  // Currency
  currencyCode: string;          // USD, EUR, OMR
  exchangeRate: number;          // To base currency (OMR)
  
  // Amounts (in transaction currency)
  subtotal: number;
  taxAmount: number;
  totalAmount: number;
  
  // Amounts (in base currency - OMR)
  subtotalBase: number;
  taxAmountBase: number;
  totalAmountBase: number;
  
  // Payment tracking
  amountPaid: number;
  amountPaidBase: number;
  balanceDue: number;
  balanceDueBase: number;
  
  // Status
  status: 'draft' | 'open' | 'partial' | 'paid' | 'overdue' | 'cancelled';
  
  // References
  salesOrderId?: string;
  shipmentId?: string;
  
  // Content
  lineItems: InvoiceLineItem[];
  notes?: string;
  terms?: string;
  attachments?: string[];
  
  // Audit
  companyId: string;
  branchId: string;
  createdBy: string;
  createdAt: Date;
  updatedAt: Date;
}

interface InvoiceLineItem {
  id: string;
  productId?: string;
  description: string;
  quantity: number;
  unitPrice: number;
  amount: number;
  taxRate?: number;
  taxAmount?: number;
}
```

### 4.2 Payment Entity
```typescript
interface Payment {
  id: string;
  paymentNumber: string;         // PAY-2024-0001
  type: 'received' | 'made';     // AR or AP
  
  // Party
  customerId?: string;           // For received
  supplierId?: string;           // For made
  
  // Payment details
  paymentDate: Date;
  paymentMethod: 'cash' | 'bank_transfer' | 'cheque' | 'card';
  referenceNumber?: string;
  
  // Currency
  currencyCode: string;
  exchangeRate: number;
  
  // Amounts
  amount: number;                // In payment currency
  amountBase: number;            // In OMR
  
  // Allocation
  allocations: PaymentAllocation[];
  unallocatedAmount: number;
  
  // Banking
  bankAccountId: string;
  
  // Audit
  companyId: string;
  branchId: string;
  createdBy: string;
  createdAt: Date;
}

interface PaymentAllocation {
  invoiceId: string;
  invoiceNumber: string;
  allocatedAmount: number;
  allocatedAmountBase: number;
}
```

### 4.3 Currency Entity
```typescript
interface Currency {
  code: string;                  // ISO 4217 (USD, EUR)
  name: string;
  symbol: string;
  decimalPlaces: number;
  isActive: boolean;
  isBaseCurrency: boolean;
}

interface ExchangeRate {
  id: string;
  fromCurrency: string;
  toCurrency: string;
  rate: number;
  effectiveDate: Date;
  source: 'auto' | 'manual';
  createdAt: Date;
}
```

| POST | `/api/finance/invoices/:id/cancel` | Cancel invoice |
